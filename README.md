# 🤖 AI Daily Reporter

> 每日 AI 资讯 + 天气 + 日程 = 你的私人晨报助手

## ✨ 功能特性

- 🔍 **AI 资讯聚合** - 自动爬取最新 AI 新闻
- 🌤️ **天气提醒** - 根据天气给出穿衣建议  
- 📅 **日程整合** - 显示今日待办事项
- 🎨 **精美报告** - Markdown 格式，支持飞书/邮件推送
- ⏰ **定时推送** - GitHub Actions 自动运行

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入你的 API keys

# 本地运行
npm run dev

# 生成今日报告
npm run generate
```
